# Balbodh Secondary School Management System

This workspace contains a production-ready scaffold for Balbodh Secondary School Management System: a React + Vite frontend and an Express + MongoDB backend with Firebase file storage support.
Quickstart (frontend):

```bash
# install frontend deps
npm install
# run dev
npm run dev
```

Backend (see `server/README.md`) — start the Express API, configure `.env` and seed a superadmin.

The template includes authentication (JWT), role-based access, file uploads to Firebase Storage, and core modules (students, teachers, classes, subjects, attendance, exams, fees, notices, assignments, library, transport, notifications). Continue implementing features, tests, and deployment configs as needed.
# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
